Руководство по настройке файла конфигурации nod32ms.conf

 Параметр: @selfupdate
 Значение: Автоматическое обновление программы, bolean
 Пример:selfupdate = true

 Параметр: @php_error_reporting
 Значение: Анолог функции error_reorting в php, может принимать как одно так несколько значений (http://ru.php.net/manual/en/function.error-reporting.php), string
 Пример: php_error_reporting = E_ALL или php_error_reporting = 0

 Параметр: @mirror
 Значение: Домен или ip адрес сервера обновлений, string
 Пример: mirror = update.eset.com

 Параметр: @timezone
 Значение: Временная зона в *nix системах, string
 Пример: timezone = Europe/Moscow

 Параметр: @username
 Значение: Имя пользователя, используется только при выключенном автопоиске ключей keys_autofind=false, string
 Пример: username = TRIAL-12345678

 Параметр: @password
 Значение: Пароль, используется только при выключенном автопоиске ключей keys_autofind=false, string
 Пример: password = ep6cuadp85

 Параметр: @keys_autofind
 Значение: Автопоиск ключей, boolean
 Пример: keys_autofind = true

 Параметр: @keys_autofind_page
 Значение: Колличество анализируемых страниц при автопоиске, integer
 Пример: keys_autofind_page = 5

 Параметр: @temp_dir
 Значение: Путь к временной директории для временных файлов, string
 Пример: temp_dir = /var/nod32ms/temp

 Параметр: @work_dir
 Значение: Путь к рабочей директории для антивирусных сигнатур, http, ftp, share, string
 Пример: work_dir = /var/www/localhost/htdocs

 Параметр: @log_dir
 Значение: Путь к директории с журналом работы скрипта, string
 Пример: log_dir = /var/log/nod32ms

 Параметр: @write_to_log
 Значение: Запись в журнал, boolean
 Пример: write_to_log = true

 Параметр: @show_last_error
 Значение: Вывод в консоль последней ошибки скрипта, boolean
 Пример: show_last_error = true

 Параметр: @update_version2_winnt
 Значение: Загрузка сигнатур 2-ой версии антивируса для WindowsNT, boolean
 Пример: update_version2_winnt = true

 Параметр: @update_version2_win98
 Значение: Загрузка сигнатур 2-ой версии антивируса для Windows98, boolean
 Пример: update_version2_win98 = true

 Параметр: @update_version2_language
 Значение: Загрузка определенных языковых локализаций для 2-ой версии антивируса, string
 Пример: update_version2_language = en или update_version2_language = en, fr

 Параметр: @update_version34_arch32
 Значение: Загрузка сигнатур 3 и 4 версии антивируса для 32-х битных систем, boolean
 Пример: update_version34_arch32 = true

 Параметр: @update_version34_arch64
 Значение: Загрузка сигнатур 3 и 4 версии антивируса для 64-х битных систем, boolean
 Пример: update_version34_arch64 = true

 Параметр: @update_version34_language
 Значение: Загрузка определенных языковых локализаций для 3 и 4 версии антивируса, string
 Пример: update_version34_language = rus или update_version34_language = rus, enu
